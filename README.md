# hubot-faker

A hubot script that does the things

See [`src/faker.coffee`](src/faker.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-faker --save`

Then add **hubot-faker** to your `external-scripts.json`:

```json
[
  "hubot-faker"
]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
